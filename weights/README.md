# Weights Folder

The trained model weights are hosted on Google Drive:

[Download weights from Google Drive](https://drive.google.com/drive/folders/1TJRZ8QAZQhUOw2FyqhALpjVVGvsmFL4m?usp=sharing)

Recommended files in this folder:

- `mobilenetv2_car_color_best.pt`
- `mobilenetv2_car_color_last.pt`
- `class_to_idx.json`
- `summary.json`

Notes:

- `mobilenetv2_car_color_best.pt` is the recommended file for inference and deployment
- `mobilenetv2_car_color_last.pt` is useful if you want to resume training
- `class_to_idx.json` contains the label mapping
- `summary.json` contains the evaluation summary for the released run
